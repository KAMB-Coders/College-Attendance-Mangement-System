College Attendance Management Project

Project Description: This project is aimed at developing a College Attendance Management Portal. This Web-Portal can be used as an attendance monitoring Portal for the college. Students/Faculty/Admin logging in may also access/search any information regarding Schedule and Student Attendance. Attendance and Schedule will be updated by faculty.

The front-end will be HTML pages with Java Script for client side validation where as all business logics will be in JSP reside at middle layer. And these layers will interact with third layer of database, which will be SQL database. The web server will be Tomcat
